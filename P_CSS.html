Información sobre CSS

¿Que es CSS?

CSS es el acrónimo de "Cascading Style Sheets" (Hojas de estilo en cascada) y se utiliza para dar estilo y formato a las páginas web. Permite controlar la apariencia visual de los elementos HTML, como el tamaño y tipo de letra, el color de fondo, la disposición de los elementos, entre otros aspectos.

Los estilos CSS se aplican mediante reglas que especifican un selector (como el nombre de una etiqueta HTML o un atributo) y un conjunto de propiedades y valores que se aplican a los elementos que cumplen con ese selector.

Los estilos CSS se pueden escribir directamente en el archivo HTML mediante etiquetas <style>, o en un archivo CSS separado que se enlaza al HTML mediante una etiqueta <link>.

Para aprender más sobre CSS, hay una gran cantidad de recursos en línea, como tutoriales, documentación y ejemplos de código.

Lista de estilos CSS

  • Color: Define el color del texto, fondo, borde, etc. Hay varias formas de especificar un color, como nombres de color predefinidos, valores hexadecimales o valores RGB.
  • Fuentes: Define la fuente del texto. Puedes especificar la familia de fuentes, el estilo, el peso, el tamaño, etc.
  • Tamaño: Define el tamaño del texto, bordes, altura de línea, etc. Los valores pueden ser especificados en píxeles, ems, porcentajes, etc.
  • Espacio: Define el espacio entre elementos, márgenes, rellenos, etc. Los valores pueden ser especificados en píxeles, ems, porcentajes, etc.
  • Posición: Define la posición de un elemento en la página, como absoluto, relativo, fijo, etc.
  • Display: Define cómo se muestra un elemento, como en línea, en bloque, como tabla, etc.
  • Animación: Define animaciones para elementos, como transiciones, rotaciones, etc.
  • Transformación: Define transformaciones para elementos, como rotaciones, escalados, etc.
  • Flexbox: Define un sistema de diseño flexible que permite organizar elementos en un contenedor con una distribución flexible y adaptable.
  • Grid: Define un sistema de diseño basado en una cuadrícula que permite organizar elementos en filas y columnas.
